Pipeline detallado

Paso a paso

1

📄 Extracción

El documento entra al sistema
  • Acepta PDF, Word (.docx), texto plano, Markdown
  • Extrae texto con posición, tamaño, color y formato
  • Detecta imágenes, tablas y estructura del documento
  • Genera un JSON normalizado para el pipeline
2

🎓 Traducción profesional

El Agente Traductor procesa cada segmento
Claude Sonnet — rápido & preciso
  • Selecciona el glosario según tipo de documento
  • Aplica procedimientos de traducción (Vinay & Darbelnet)
  • Verifica falsos amigos automáticamente
  • Respeta reglas gramaticales del español formal
  • Agrega notas del traductor cuando es necesario
3

🔎 Revisión estricta

El Agente Revisor verifica y corrige
Claude Opus — el modelo más analítico
  • Revisa en 7 dimensiones de calidad profesional
  • Clasifica cada issue: crítico, mayor o menor
  • Corrige automáticamente cada error encontrado
  • Calcula score numérico de calidad
  • Decide si está listo para certificación
4

✔ Output final

Listo para la traductora pública
  • Texto revisado — Final, corregido y consistente
  • Reporte de calidad — Score + detalle de cada issue
  • Score ≥ 0.95 y 0 errores críticos = listo para firmar
  • Si hay issues, el reporte explica cada uno con justificación
